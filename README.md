# <span style="font-size: 2em;">AZSwap</span>

## <span style="font-size: 1.5em;">Overview</span>
**AZSwap** is a decentralized exchange (DEX) application designed to provide users with a seamless and efficient way to swap cryptocurrency tokens. Users can connect their MetaMask wallet, select from a list of tokens, and perform swaps with accurate pricing. The application offers slippage settings for enhanced control over transactions and maintains a minimalist user interface for ease of use.

## <span style="font-size: 1.5em;">Features</span>
- **MetaMask Integration**: Easily connect your MetaMask wallet to start swapping tokens.
- **Token Selection**: Choose from a dropdown menu of available tokens for swapping.
- **Accurate Pricing**: Real-time token prices fetched via APIs ensure accurate and fair swaps.
- **Slippage Settings**: Select from predefined slippage percentages (0.5%, 2.5%, 5.0%) to control your transaction tolerance.
- **Minimalist UI**: A clean and straightforward interface for an enhanced user experience.
